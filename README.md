# Tongram Messenger

Tongram is a messaging application built on Telegram's open-source codebase, enhanced with AI-powered features.

## Features
- AI Writing Assistant (grammar, tone, translation)
- Smart Folders (AI-powered chat organization)
- Chat Summary (AI summarizes long conversations)
- Custom UI with distinct branding
